Tutorial: Clustering the Iris Dataset via K-Means

Tuesday, January 19, 2016 @ 05:30 PM in DOW 2166

Interested in Machine Learning? Don't know where to start? Never fear!!

This Tuesday, join Aaron Pollack and Samuel Tenka for an overview of machine learning and a brief survey of its various applications. Then we'll dive into a hands-on Machine Learning project, in which we'll:

  1. visualize a dataset of flower measurements,
  2. wonder how to group the data-points into species,
  3. formalize this wonder as a problem in "Clustering",
  4. invent a fully-automatic Clustering algorithm,
  5. implement that algorithm using standard packages, and
  6. analyze algorithm performance.

By the end of our session, you'll have:

  • A Mental map of Machine Learning's lay of the land.
  • Experience using standard packages to analyze real data.
  • Enthusiasm to participate in more of MSAIL's activities!

The problem is computationally difficult (NP-hard); however, there are efficient heuristic algorithms that are commonly employed and converge quickly to a local optimum. These are usually similar to the expectation-maximization algorithm for mixtures of Gaussian distributions via an iterative refinement approach employed by both algorithms. Additionally, they both use cluster centers to model the data; however, k-means clustering tends to find clusters of comparable spatial extent, while the expectation-maximization mechanism allows clusters to have different shapes.
